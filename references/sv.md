## Stavningsregler

### å, ä, ö — aldrig ersätta med aa, ae, oe

Det vanligaste AI-felet är att skriva `aa`, `ae` eller `oe` istället för de svenska bokstäverna.

| Fel | Rätt |
|-----|------|
| `aar` | `år` |
| `alder` | `ålder` |
| `ater` | `åter` |
| `anga` | `ånga` |
| `aven` | `även` |
| `aldre` | `äldre` |
| `andra` | `ändra` |
| `ange` | `änge` / `äng` |
| `oppna` | `öppna` |
| `over` | `över` |
| `osta` | `östa` / `öst` |
| `ora` | `öra` |

### Stor/liten bokstav

Svenska använder stor bokstav i början av mening och för egennamn, men INTE för veckodagar, månader, nationaliteter eller språk (till skillnad från engelskan):

| Fel (engelska mönster) | Rätt (svenska) |
|------------------------|----------------|
| `Måndag` (i mitten av mening) | `måndag` |
| `Januari` (i mitten av mening) | `januari` |
| `Svensk` (adjektiv) | `svensk` |
| `Engelska` (språk) | `engelska` |

## Vanliga AI-fel på svenska

- Skriva `aa`, `ae`, `oe` istället för `å`, `ä`, `ö`
- Kopiera engelskt bruk av stora bokstäver för veckodagar och månader
- Förväxla `där`/`dar`, `när`/`nar`, `för`/`for`
- Utelämna umlaut: `koppa` istället för `köpa`, `gora` istället för `göra`
