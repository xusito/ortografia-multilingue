# Deutsch — Rechtschreibreferenz

## Offizielle Wörterbücher
- **Primär**: https://www.duden.de
- **Sekundär**: https://www.dwds.de (Digitales Wörterbuch der deutschen Sprache)

Bei Zweifel an einer Schreibweise zuerst im Duden nachschlagen. Wenn die ursprüngliche Form korrekt ist, nicht ändern.

---

## Besondere Zeichen des Deutschen

`ä` `ö` `ü` `Ä` `Ö` `Ü` `ß`

---

## Umlaute: ä, ö, ü

Umlaute sind eigene Buchstaben — keine Abkürzungen. Die Schreibweisen `ae`, `oe`, `ue` sind nur als technischer Ersatz akzeptabel (z.B. in E-Mail-Adressen), nicht in normalem Text.

| Fehler | Korrektur |
|---|---|
| `Mutter` (korrekt) | `Mütter` (Plural) |
| `Baume` | `Bäume` |
| `konnen` | `können` |
| `mochte` | `möchte` |
| `uber` | `über` |
| `Tur` | `Tür` |
| `Buro` | `Büro` |
| `Fuhre` | `Führe` |
| `zuruck` | `zurück` |
| `naturlich` | `natürlich` |
| `schon` (gemeint: schön) | `schön` |
| `groß` (korrekt) | `groß` ✓ |

## Eszett (ß) und ss

Das `ß` (scharfes S) steht nach langen Vokalen und Diphthongen. Nach kurzen Vokalen steht `ss`. Seit der Rechtschreibreform 1996 gelten klare Regeln:

**ß nach langem Vokal oder Diphthong:**

| Fehler | Korrektur |
|---|---|
| `Strasse` | `Straße` |
| `heissen` | `heißen` |
| `weiss` | `weiß` |
| `gruss` | `Gruß` |
| `Fuss` | `Fuß` |
| `Masse` (Maß) | `Maß` (= measure) |

**ss nach kurzem Vokal:**

| Fehler | Korrektur |
|---|---|
| `muß` (nach kurz. Vokal) | `muss` |
| `aß` → korrekt ✓ | (nach langem a) |
| `Fluss` ✓ | (nach kurzem u) |

**Schweiz**: In der Schweiz wird `ß` nie verwendet — immer `ss`. Beide Varianten sind korrekt, solange ein Text konsistent ist.

## Großschreibung von Substantiven

Im Deutschen werden **alle Substantive** großgeschrieben. Dies ist einer der häufigsten Fehler in KI-Texten.

| Fehler | Korrektur |
|---|---|
| `das haus` | `das Haus` |
| `ein auto` | `ein Auto` |
| `die arbeit` | `die Arbeit` |
| `mit freude` | `mit Freude` |
| `beim lesen` | `beim Lesen` |
| `das schreiben` | `das Schreiben` |
| `die qualitat` | `die Qualität` |
| `eine moglichkeit` | `eine Möglichkeit` |

Adjektive und Verben bleiben klein, außer am Satzanfang oder als Substantivierungen.

## Häufige Rechtschreibfehler

| Fehler | Korrektur |
|---|---|
| `das` (Konjunktion) | `dass` (Konjunktion "that") — `das` = Artikel/Pronomen |
| `seit` | `seid` (ihr seid = you are) — `seit` = since/for (temporal) |
| `wider` | `wieder` (again) — `wider` = against |
| `Standart` | `Standard` |
| `Adresse` ✓ | (Doppel-s, kein ß) |
| `kommt` ✓ | (kein Umlaut) |

---

## Häufige KI-Fehler auf Deutsch

- Umlaute als `ae`/`oe`/`ue` schreiben statt `ä`/`ö`/`ü`
- `ß` und `ss` verwechseln
- Substantive kleinschreiben
- `das`/`dass` verwechseln
- `seit`/`seid` verwechseln
- Großschreibung am Satzanfang vergessen
