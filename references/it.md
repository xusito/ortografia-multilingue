## Caratteri speciali dell'italiano

`à` `è` `é` `ì` `í` `ò` `ó` `ù` `ú`

L'italiano usa principalmente l'accento grave (`), con alcune eccezioni in accento acuto (´) su `é` e `ó` in parole specifiche.

---

## Regole di accentuazione

### Parole tronche (ossitone): accent obbligatorio sull'ultima sillaba

Parole che terminano con vocale accentata:

| Errore | Correzione |
|---|---|
| `citta` | `città` |
| `caffe` | `caffè` |
| `universita` | `università` |
| `attivita` | `attività` |
| `qualita` | `qualità` |
| `capacita` | `capacità` |
| `liberta` | `libertà` |
| `virtu` | `virtù` |
| `tribu` | `tribù` |
| `piu` | `più` |
| `gia` | `già` |
| `li` | `lì` (avverbio di luogo) |
| `si` | `sì` (affermazione) |

### Monosillabi con accento distintivo

Questi monosillabi cambiano significato con/senza accento:

| Senza accento | Con accento | Significato |
|---|---|---|
| `e` (congiunzione: "e") | `è` (verbo essere: "è") | and / is |
| `la` (articolo) | `là` (avverbio: "there") | the / there |
| `li` (pronome) | `lì` (avverbio: "there") | them / there |
| `si` (pronome) | `sì` (affermazione: "yes") | himself / yes |
| `se` (congiunzione) | `sé` (pronome riflessivo) | if / himself |
| `ne` (particella) | `né` (né...né: "neither") | of it / neither |
| `da` (preposizione) | `dà` (verbo dare: 3a pers.) | from / gives |
| `te` (pronome) | `tè` (bevanda) | you / tea |

## Errori comuni nelle traduzioni IA

- Omettere l'accento sulle parole tronche (`citta` → `città`, `caffe` → `caffè`)
- Confondere `e`/`è`, `si`/`sì`, `ne`/`né`, `da`/`dà`
- Usare apostrofo curvo invece di dritto
- Usare accento acuto dove serve grave (es: `café` in stile francese invece di `caffè`)
- Scrivere `po'` senza apostrofo (`po` → `po'`, troncamento di "poco")

---

## Apostrofo: sempre dritto

Usare `'` (dritto), mai `'` né `'` (curvi):
- `dell'uomo` ✓ — `dell'uomo` ✗

---

## Doppie consonanti

L'italiano usa le doppie consonanti con frequenza. Errori comuni di IA:

| Errore | Correzione |
|---|---|
| `acomodare` | `accomodare` |
| `conección` | `connessione` |
| `posibile` | `possibile` |
| `necesario` | `necessario` |
| `afinare` | `affinare` |
| `oferta` | `offerta` |
| `efeto` | `effetto` |
