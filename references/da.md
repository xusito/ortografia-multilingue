## Danske specialtegn

`å` `æ` `ø`

## Stavningsregler

### å, æ, ø — aldrig erstattes med aa, ae, oe

Det hyppigste AI-fejl er at bruge `aa`, `ae` eller `oe` i stedet for de danske bogstaver.

**Vigtigt om `aa`**: `aa` er den historiske stavning af `å` og bruges stadig i visse egennavne (f.eks. `Aarhus` er officielt, men `Århus` er også acceptabelt). I almindelige ord bruges altid `å`.

| Fejl | Rigtigt |
|------|---------|
| `aar` | `år` |
| `aaben` | `åben` |
| `aand` | `ånd` |
| `kaere` | `kære` |
| `laese` | `læse` |
| `baere` | `bære` |
| `saette` | `sætte` |
| `vaere` | `være` |
| `groen` | `grøn` |
| `hoere` | `høre` |
| `goere` | `gøre` |
| `stoerre` | `større` |
| `oe` (bogstav) | `ø` |

### Hyppige ord med æ, ø, å

| Fejl | Rigtigt |
|------|---------|
| `for` (preposition) | `for` ✓ / `før` (before) |
| `gar` | `går` (goes) |
| `star` | `står` (stands) |
| `nar` | `når` (when) |
| `her` | `her` ✓ (here) — `hær` (army) |
| `der` | `der` ✓ (there/who) — `dær` ✗ |
| `og` ✓ | (and — ingen accent) |
| `at` ✓ | (to/that — ingen accent) |
| `det` ✓ | (it/that) |

### Store/små bogstaver

Dansk bruger stort begyndelsesbogstav i sætningens begyndelse og i egennavne, men IKKE for ugedage, måneder og nationaliteter (modsat engelsk):

| Fejl (engelsk mønster) | Rigtigt (dansk) |
|------------------------|-----------------|
| `Mandag` (midt i sætning) | `mandag` |
| `Januar` (midt i sætning) | `januar` |
| `Dansk` (adjektiv) | `dansk` |
| `Engelsk` (sprog) | `engelsk` |

## Hyppige AI-fejl på dansk

- Skrive `aa`, `ae`, `oe` i stedet for `å`, `æ`, `ø`
- Forveksle `før`/`for`, `når`/`nar`, `høre`/`hore`
- Kopiere engelsk brug af store bogstaver for ugedage og måneder
- Udelade accent: `laese` i stedet for `læse`, `goere` i stedet for `gøre`

## Tegnsætning

| Regel | Rigtigt | Forkert |
|---|---|---|
| Anførselstegn | `»tekst«` eller `"tekst"` | `'tekst'` |
| Intet mellemrum før `:` `;` | `eksempel: tekst` | `eksempel : tekst` |
| Tankestreg med mellemrum | `tekst – tekst` | `tekst-tekst` (som tankestreg) |
| Decimalkomma | `3,14` | `3.14` |
| Tusinder | `1.000` | `1,000` |
