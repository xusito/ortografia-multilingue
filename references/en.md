# English — Spelling Reference

## Official Dictionaries
- **Primary (US)**: https://www.merriam-webster.com
- **Primary (UK)**: https://www.oxfordlearnersdictionaries.com
- **Style guide**: https://www.apstylebook.com (journalism) / Chicago Manual of Style (publishing)

When in doubt about a word, look it up before correcting. If the original form is correct, leave it.

**Note on variants**: flag US/UK spelling inconsistencies within the same document (e.g., mixing `color`/`colour`), but both are correct on their own.

## Homophones and confusables

These sound similar but have different meanings and spellings:

| Wrong use | Correct form | Rule |
|---|---|---|
| `it's cold` (possessive) | `its cold` | `its` = possessive; `it's` = it is |
| `their going` | `they're going` | `they're` = they are; `their` = possessive |
| `there car` | `their car` | `their` = possessive |
| `your welcome` | `you're welcome` | `you're` = you are |
| `effect` (verb) | `affect` (verb) | `affect` = to influence; `effect` = result (noun) |
| `then` (comparison) | `than` (comparison) | `than` = for comparisons |
| `loose` (verb: to free) | `lose` | `lose` = to not win / misplace |
| `compliment` (free) | `complement` | `complement` = goes well with |
| `principal` (rule) | `principle` | `principal` = main person/thing |

## Apostrophes

| Error | Correction | Rule |
|---|---|---|
| `it's` (possessive) | `its` | No apostrophe for `its` possessive |
| `the dog's are here` | `the dogs are here` | No apostrophe for plain plurals |
| `1990's music` | `1990s music` | No apostrophe in decade plurals |
| `your` (you are) | `you're` | Contraction needs apostrophe |

## Common AI spelling errors in English

- Missing double letters (`necesary`, `occurance`, `accomodate`)
- `its`/`it's` confusion
- `their`/`there`/`they're` confusion
- `affect`/`effect` confusion
- `separate` misspelled as `seperate`
- `definitely` misspelled as `definately`

## Punctuation

| Rule | Correct | Incorrect |
|---|---|---|
| US: punctuation inside quotes | `He said "yes."` | `He said "yes".` |
| UK: punctuation outside quotes | `He said "yes".` | `He said "yes."` |
| Apostrophe straight | `it's` (straight) | `it's` (curly) |
| No space before `:` `;` | `example: text` | `example : text` |
| Decimal | `3.14` | `3,14` |
| Thousands | `1,000` | `1.000` |
| Em dash (no spaces) | `text—text` | `text — text` |
| Hyphen vs em dash | `well-known` (hyphen) vs `text—text` (em dash) | mixing both |
