# Català — Referència ortogràfica

## Diccionaris oficials
- **Primari**: https://dlc.iec.cat (DIEC — Diccionari de la Llengua Catalana)
- **Secundari**: https://optimot.gencat.cat (servei de consultes lingüístiques)

Davant el dubte sobre una paraula, consulta el DIEC abans de corregir. Si la forma original és correcta, no la modifiquis.

**Varietat**: català central (Barcelona). No valencià ni balear.

---

## Regles d'accentuació

El català té un sistema d'accentuació propi, diferent de l'espanyol. L'error més freqüent és aplicar regles espanyoles al català.

### Accent greu (`) — vocals obertes: à, è, ò

| Error | Correcció |
|---|---|
| `catala` | `català` |
| `tambe` | `també` |
| `pero` | `però` (conjunció adversativa) |
| `es` (verb ser) | `és` |
| `interes` | `interès` |
| `angles` | `anglès` |
| `mes` (quan = "más") | `més` |
| `allo` | `allò` |
| `aixo` | `això` |
| `perque` | `perquè` (conjunció) |

### Accent agut (´) — vocals tancades: é, í, ó, ú

| Error | Correcció |
|---|---|
| `raco` | `racó` |
| `canto` (passat 3a p.) | `cantó` |

### Paraules que NO porten accent en català

Error habitual: afegir accents per analogia amb l'espanyol. Formes correctes sense accent:

- `reformes` (no `reformés`)
- `necessita` (no `necessíta`)
- `empresa`, `servei`, `porta`, `casa`, `persona`, `sistema`, `feina`
- `instal·lacions` (no `instal·lacións` — el plural no porta accent)

---

## Punt volat (·) — l geminada

La doble l sonora en català s'escriu `l·l`, mai `ll` ni `l.l`:

| Error | Correcció |
|---|---|
| `collegi` | `col·legi` |
| `illegal` | `il·legal` |
| `collaborar` | `col·laborar` |
| `intellecte` | `intel·lecte` |
| `intelligencia` | `intel·ligència` |
| `installacio` | `instal·lació` |

Nota: `tall`, `ball`, `grall` porten `ll` normal (so diferent, sense punt volat).

## Ce trencada (ç) i dièresi

| Error | Correcció |
|---|---|
| `forca` | `força` |
| `placa` | `plaça` |
| `anca` | `ança` |
| `veina` | `veïna` |
| `raim` | `raïm` |
| `questio` | `qüestió` |
| `frequent` | `freqüent` |

---

## Apòstrof: sempre recte

Usar `'` (recte), mai `'` ni `'` (corbs):
- `l'empresa` ✓ — `l'empresa` ✗

---

## Errors freqüents en traduccions IA

- Afegir accent al plural de paraules acabades en -ció: `instal·lacións` → `instal·lacions`
- Posar accent en lloc equivocat: `comúnicació` → `comunicació`
- Ometre punt volat: `col.legi`, `collegi` → `col·legi`
- Ometre ce trencada: `placa` → `plaça`
- Usar apòstrof corb en lloc de recte
- Vocabulari IA prohibit: `vibrant`, `crucial`, `pivotal`, `testament`, `delve`, `destacant`, `serveix com`, `s'erigeix`, `encarna`, `en el cor de`, `subratlla`, `ressalta`, `embarcar-se`, `endinsar-se`, `enclavat`, `fomentar`, `intricat`, `tapís`, `entrellaçar`
