- **Primario**: https://rae.es (DLE — Diccionario de la Lengua Española)

Ante duda sobre una palabra, búscala en el DLE antes de corregir. Si la forma original es correcta, no la modifiques.

### Interrogativas y exclamativas: siempre con tilde

`qué`, `cómo`, `cuándo`, `dónde`, `quién`, `cuánto`, `cuál`, `cuántos`

Llevan tilde también en preguntas indirectas: "no sé cómo hacerlo", "dime cuándo llegas".

### Casos con distinción de significado

| Sin tilde | Con tilde |
|---|---|
| `el` (artículo) | `él` (pronombre personal) |
| `tu` (posesivo) | `tú` (pronombre personal) |
| `mi` (posesivo) | `mí` (pronombre personal) |
| `se` (pronombre) | `sé` (verbo saber/ser) |
| `si` (condicional) | `sí` (afirmación / pronombre reflexivo) |
| `mas` (conjunción: "pero") | `más` (adverbio de cantidad) |
| `te` (pronombre) | `té` (bebida) |
| `de` (preposición) | `dé` (verbo dar) |
| `solo` (adjetivo) | `solo` (adverbio, sin tilde desde RAE 2010) |

## Vocabulario IA a evitar — con sustituciones

| Palabra IA | Alternativas naturales |
|---|---|
| `crucial` | importante, decisivo, determinante, clave |
| `fundamental` | básico, necesario, esencial, central |
| `pivotal` | decisivo, de peso, determinante |
| `vibrante` | activo, dinámico, con vida propia |
| `rico` (figurativo) | variado, amplio, con mucho donde elegir |
| `innovador` | nuevo, distinto, diferente |
| `revolucionario` | que cambia las reglas, sin precedentes |
| `exhaustivo` | completo, detallado, minucioso |
| `sin fisuras` | fluido, sin interrupciones, continuo |
| `robusto` | sólido, fiable, bien construido |
| `potenciar` | mejorar, reforzar, aumentar |
| `aprovechar` | usar, sacar partido de, explotar |
| `paisaje` (abstracto) | sector, entorno, situación actual |
| `testimonio de` | muestra de, refleja, demuestra |
| `subrayar` | recordar, insistir en, señalar |
| `destacar` (surutilizado) | mencionar, señalar, indicar |
| `en el corazón de` | en el centro de, dentro de |
| `es importante señalar` | hay que decir, conviene saber |
| `vale la pena destacar` | merece la pena mencionar |
| `sumergirse en` | explorar, conocer, adentrarse en |

## Errores frecuentes en traducciones IA

- Omitir tildes en esdrújulas (`tecnico`, `basico`, `multiple`)
- Omitir tilde en interrogativas (`que`, `como`, `cuando`)
- Confundir `tu`/`tú`, `el`/`él`, `si`/`sí`
- Escribir `mas` sin tilde cuando significa "más"

## Puntuación

| Regla | Correcto | Incorrecto |
|---|---|---|
| Apertura interrogativa | `¿Cómo funciona?` | `Como funciona?` |
| Apertura exclamativa | `¡Perfecto!` | `Perfecto!` |
| Comillas | `«texto»` o `"texto"` | `'texto'` |
| Sin espacio antes de `:` `;` | `ejemplo: texto` | `ejemplo : texto` |
| Decimal | `3,14` | `3.14` |
| Miles | `1.000` | `1,000` |
