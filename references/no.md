# Norsk — Stavningsreferanse

## Offisielle ordbøker
- **Primær (Bokmål)**: https://ordbokene.no (Bokmålsordboka)
- **Primær (Nynorsk)**: https://ordbokene.no (Nynorskordboka)
- **Sekundær**: https://naob.no (Det Norske Akademis ordbok)

Ved tvil om stavning, slå opp ordet i Bokmålsordboka før retting. Hvis originalformen er korrekt, ikke endre.

**Varianter**: norsk har to offisielle skriftformer — Bokmål (mest brukt, ~85%) og Nynorsk. Vær konsistent innenfor samme tekst.

## Norske spesialtegn

`å` `æ` `ø`

## Stavningsregler

### å, æ, ø — aldri erstate med aa, ae, oe

Det vanligste AI-feilen er å bruke `aa`, `ae` eller `oe` i stedet for de norske bokstavene.

**Merk om `aa`**: `aa` er den historiske formen av `å` og brukes ikke i moderne norsk (unntatt i noen egennamn som `Aalborg` i dansk kontekst). I norske ord brukes alltid `å`.

| Feil | Riktig (Bokmål) |
|------|-----------------|
| `aar` | `år` |
| `aapen` | `åpen` |
| `aand` | `ånd` |
| `kaere` | `kjære` |
| `laese` | `lese` |
| `baere` | `bære` |
| `saette` | `sette` |
| `vaere` | `være` |
| `groen` | `grønn` |
| `hoere` | `høre` |
| `goere` | `gjøre` |
| `stoerre` | `større` |

### Vanlige ord med æ, ø, å

| Feil | Riktig |
|------|--------|
| `gar` | `går` (goes) |
| `star` | `står` (stands) |
| `nar` | `når` (when) |
| `for` | `for` ✓ / `før` (before) |
| `og` ✓ | (and — ingen aksent) |
| `er` ✓ | (is/are) |
| `ikke` ✓ | (not) |
| `sa` | `så` (so/then) |
| `ma` | `må` (must) |
| `fa` | `få` (few/get) |

### Bokmål vs Nynorsk — vanlige forskjeller

| Bokmål | Nynorsk | Betydning |
|--------|---------|-----------|
| `ikke` | `ikkje` | not |
| `jeg` | `eg` | I |
| `hva` | `kva` | what |
| `noe` | `noko` | something |
| `gjøre` | `gjere` | to do |
| `lese` | `lesa` | to read |
| `skrive` | `skriva` | to write |

### Store/små bokstaver

Norsk bruker stor bokstav i setningens begynnelse og i egennavn, men IKKE for ukedager, måneder og nasjonaliteter (i motsetning til engelsk):

| Feil (engelsk mønster) | Riktig (norsk) |
|------------------------|----------------|
| `Mandag` (midt i setning) | `mandag` |
| `Januar` (midt i setning) | `januar` |
| `Norsk` (adjektiv) | `norsk` |
| `Engelsk` (språk) | `engelsk` |

## AI-vokabular å unngå — med erstatningsord

| AI-ord | Naturlige alternativer |
|---|---|
| `avgjørende` (overbrukt) | viktig, sentral, betydningsfull |
| `grunnleggende` | grunnleggende, nødvendig, sentral |
| `banebrytende` | ny, nyskapende, original |
| `livlig` (vibrant) | aktiv, levende, dynamisk |
| `rik` (figurativt) | variert, bred, innholdsrik |
| `sømløs` | smidig, uavbrutt, friksjonsfri |
| `robust` | stabil, pålitelig, velbygget |
| `landskap` (abstrakt) | bransje, miljø, sektor |
| `vitnesbyrd om` | viser, beviser, er et tegn på |
| `understreke` | betone, påpeke, presisere |
| `fremheve` | nevne, vise, presentere |
| `forme` | bygge, definere, påvirke |
| `i hjertet av` | i sentrum av, midt i |
| `det er viktig å merke seg` | merk at, viktig: |
| `i dagens verden` | i dag, nå, for tiden |
| `dykke ned i` | utforske, undersøke, sette seg inn i |

## Vanlige AI-feil på norsk

- Skrive `aa`, `ae`, `oe` i stedet for `å`, `æ`, `ø`
- Blande Bokmål og Nynorsk i samme tekst
- Kopiere engelsk bruk av store bokstaver for ukedager og måneder
- Utelate aksent: `gar` i stedet for `går`, `star` i stedet for `står`
- Forkorte `ikke` til `ikke` ✓ (riktig) men bruke `ikkje` i Bokmål-tekst

## Tegnsetting

| Regel | Riktig | Feil |
|---|---|---|
| Anførselstegn | `«tekst»` (Bokmål) eller `"tekst"` | `'tekst'` |
| Intet mellomrom før `:` `;` | `eksempel: tekst` | `eksempel : tekst` |
| Tankestrek med mellomrom | `tekst – tekst` | `tekst-tekst` (som tankestrek) |
| Desimalkomma | `3,14` | `3.14` |
| Tusener | `1 000` (mellomrom) | `1,000` |
