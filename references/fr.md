# Français — Référence orthographique

## Dictionnaires officiels
- **Primaire**: https://www.academie-francaise.fr/le-dictionnaire
- **Secondaire**: https://www.cnrtl.fr (Centre National de Ressources Textuelles et Lexicales)
- **Pratique**: https://www.larousse.fr/dictionnaires/francais

En cas de doute sur un mot, consulter le dictionnaire avant de corriger. Si la forme originale est correcte, ne pas la modifier.

---

## Caractères spéciaux du français

`é` `è` `ê` `ë` `à` `â` `ù` `û` `ü` `î` `ï` `ô` `ç` `œ` `æ`

---

## Règles d'accentuation

### Accent aigu (´): uniquement sur le `e`
Indique un son fermé [e]:

| Erreur | Correction |
|---|---|
| `ete` | `été` |
| `cafe` | `café` |
| `ecole` | `école` (→ accent grave ici, voir dessous) |
| `etat` | `état` (→ accent grave) |
| `cle` | `clé` |
| `verite` | `vérité` |
| `precis` | `précis` |
| `generale` | `générale` |

### Accent grave (`): sur `e`, `a`, `u`
Indique un `e` ouvert [ɛ], distingue des homophones pour `à`/`a` et `où`/`ou`:

| Erreur | Correction | Note |
|---|---|---|
| `tres` | `très` | adverbe |
| `apres` | `après` | |
| `deja` | `déjà` | |
| `a` | `à` | préposition (vs `a` = verbe avoir) |
| `ou` | `où` | adverbe de lieu (vs `ou` = conjonction) |
| `la` | `là` | adverbe de lieu (vs `la` = article) |

### Accent circonflexe (^): sur `â`, `ê`, `î`, `ô`, `û`
Marque souvent une voyelle longue ou une lettre historique disparue:

| Erreur | Correction |
|---|---|
| `foret` | `forêt` |
| `fete` | `fête` |
| `baton` | `bâton` |
| `tete` | `tête` |
| `hote` | `hôte` |
| `sur` | `sûr` (adjectif: certain) — `sur` = préposition |
| `du` | `dû` (participe de devoir) — `du` = article contracté |

### Tréma (¨): `ë`, `ï`, `ü`
Indique que deux voyelles consécutives se prononcent séparément:

| Erreur | Correction |
|---|---|
| `noel` | `Noël` |
| `naive` | `naïve` |
| `ambigue` | `ambiguë` |

### Cédille (ç)
`ç` devant `a`, `o`, `u` pour donner le son [s]:

| Erreur | Correction |
|---|---|
| `ca` | `ça` |
| `garcon` | `garçon` |
| `francais` | `français` |
| `facade` | `façade` |
| `reca` | `reçu` |

## Majuscules avec accents

En français, les majuscules portent aussi les accents. Erreur fréquente des IA: les supprimer.

| Erreur | Correction |
|---|---|
| `Etat` | `État` |
| `Ete` | `Été` |
| `Ecole` | `École` |
| `A` (préposition) | `À` |

---

## Ligatures

| Erreur | Correction |
|---|---|
| `oeuvre` | `œuvre` |
| `coeur` | `cœur` |
| `soeur` | `sœur` |
| `noeud` | `nœud` |
| `oeuf` | `œuf` |

(Acceptables sans ligature dans les contextes numériques, mais la ligature est préférable.)

---

## Apostrophe: toujours droite

Utiliser `'` (droit), jamais `'` ni `'` (courbes):
- `l'homme` ✓ — `l'homme` ✗

---

## Vocabulaire IA à éviter (style artificiel)

Ces mots et expressions apparaissent de façon excessive dans les textes générés par IA. Les signaler et proposer des alternatives naturelles :

`crucial` · `fondamental` · `incontournable` · `essentiel` (surutilisé) · `vibrant` · `riche` (figuratif) · `mettre en lumière` · `souligner` · `témoigne de` · `témoignage de` · `paysage` (abstrait : "le paysage numérique") · `au cœur de` · `façonner` · `stimuler` · `mettre en avant` · `naviguer` (abstrait) · `s'inscrit dans` · `dans le cadre de` · `il convient de noter` · `il est important de souligner` · `plonger dans`

---

## Erreurs fréquentes dans les traductions IA

- Omettre les accents sur les majuscules (`Etat` → `État`)
- Confondre accent aigu/grave sur `e` (`étre` → `être`)
- Omettre la cédille (`francais` → `français`, `ca` → `ça`)
- Confondre `à`/`a`, `où`/`ou`, `là`/`la`, `sûr`/`sur`, `dû`/`du`
- Omettre le tréma (`naive` → `naïve`)
- Oublier les ligatures œ/æ
